> In this project; it is required to design a Collision Avoidance System.
> Different UML Diagrams were used to clearly describe the system and subsystems.
> TTool was used for a complete modeling and verification of the system.
> This is a Baremetal Software that receives a signal representing the distance to the nearest obstacle from an Ultrasonic Sensor, and accordingly it sends an ON/OFF signal to a DC-Motor.
> All {*.h & *.c} files were created and designed by myself.
> The Software was developed using Eclipse-IDE and was auto built using MinGW GCC Toolchain.
> The Software was debugged using Eclipse-IDE.
> A Test Run was carried out to verify and validate the software, and the results were attached.
