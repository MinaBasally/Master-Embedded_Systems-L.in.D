> This Software is a Student Management System(Student Information System).
> The Software creates a Students Database in the form of a Statically-Allocated Array with a preset maximum number of entries.
> All {*.h & *.c} files were created and designed by myself.
> The Software was developed using Eclipse-IDE and was auto built using MinGW GCC Toolchain.


> The Student Management System includes the following APIs :

	### 1. add_student_from_file()
 API Function: /* Add students from A Text File */

	### 2. add_student_manually()
 API Function: /* Add students manually */

	### 3. find_RN()
 API Function: /* Find students by Their Roll Numbers and Print them */

	### 4. find_FN()
 API Function: /* Find students by Their First Names and Print them */

	### 5. find_course()
 API Function: /* Find and Print All the students who registered a certain course */

	### 6. tot_st()
 API Function: /* Print The Total Number of students in The Database */

	### 7. del_st()
 API Function: /* Find students by Their Roll Numbers and Delete them */

	### 8. upd_st()
 API Function: /* Find students by Their Roll Numbers and Update them */

	### 9. show_st()
 API Function: /* Print All the students in The Database */
